## Installation

<br/><br/>

Add `@nuxtjs/xxx` dependency to your project:

<br/><br/>

<code-group>
  <code-block label="Yarn" active>

```bash
yarn add @nuxtjs/xxx
```

  </code-block>
  <code-block label="NPM">

```bash
npm install @nuxtjs/xxx
```

  </code-block>
</code-group>

<br/><br/>

Then, add `@nuxtjs/xxx` to the `modules` section of `nuxt.config.js`:

<br/><br/>

```js[nuxt.config.js]
{
  modules: [
    '@nuxtjs/xxx'
  ],
  xxx: {
    // Options
  }
}
```
